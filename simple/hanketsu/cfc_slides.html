CFC税制（外国子会社合算税制）概要
タックスヘイブン対策税制の基本構造
1/15
制度趣旨
目的：税負担の著しく低い国・地域（タックスヘイブン）に子会社等を設立し、所得を留保することによる租税回避への対処
  • 昭和53年（1978年）導入
  • 税負担の実質的公平を図る
  • 海外投資の不当な阻害を回避するため適用除外規定あり

判旨の分析 - 国家主権と課税権
5/15
最高裁判旨（要旨）
「一般に、自国における税負担の公平性や中立性に有害な影響をもたらす可能性のある他国の制度に対抗する手段として、いわゆるタックス・ヘイブン対策税制を設けることは、国家主権の中核に属する課税権の内容に含まれるものと解される。」
「租税条約その他の国際約束等によってこのような税制を設ける我が国の権能が制約されるのは、当該国際約束におけるその旨の明文規定その他の十分な解釈上の根拠が存する場合でなければならない」
「日星租税条約7条1項は、いわゆる法的二重課税を禁止するにとどまるものであって、同項が禁止又は制限している行為は、一方の締約国の企業に対する他方の締約国の課税権の行使に限られる」
判断の論理構造
大前提
CFC税制は国家主権の課税権
→
小前提
条約に明文の制約規定なし
→
結論
CFC税制は条約非違反
補充的理由：
CFC税制は、あくまで「内国法人/居住者」への課税であり、シンガポール法人の利得に「直接」課税するものではない
制度趣旨との整合性
CFC税制の合理性認定
  • 税負担の公平性追求という正当な目的
  • 独立企業としての実体があれば適用除外
  • 経済合理性ある海外投資は阻害しない
  • 全体として合理的な制度設計
所得税版の特殊性
  • 法人税と異なり外国税額控除なし
  • しかしこれは法人税との均衡から合理的

判旨の分析 - 外国法人税の意義
8/15
最高裁判旨（要旨）
「法人税法69条1項にいう『外国法人税』とは、外国の法令により法人の所得を課税標準として課される税をいうものであり...その税が任意に選択可能な制度の下で課されるものであっても、その適用を選択した場合に課される税が所得を課税標準として課されるものである限り、『外国法人税』に該当する」
「ガーンジー島の国際課税資格制度に基づく課税は、ガーンジー島内に源泉を有する所得を課税標準として課される税であり、法人税法69条1項にいう『外国法人税』に該当する」
判断の論理構造
要件判断
外国の法令による税○ ガーンジーの所得税法に基づく
法人に対する税○ 法人を納税義務者とする
所得を課税標準○ 島内源泉所得に対し20%課税
強制性の要否不要（選択制でも可）
ポイント：形式的に選択制であっても、選択後に課される税が「所得を課税標準とする税」であれば外国法人税に該当
下級審との相違
東京高裁（原審）の判断
  • ITR制度は選択制であり任意
  • 全所得課税を避けるための制度
  • 実質的には所得課税ではない
  • → 外国法人税に該当しない
最高裁の逆転
  • 選択制か否かは決定的でない
  • 課される税の性質を実質判断
  • 島内源泉所得への20%課税は所得課税
  • → 外国法人税に該当する

判旨の分析 - 「主たる事業」の判断基準
11/15
最高裁判旨（要旨）
「外国子会社の『事業』がいかなるものであるかは、当該外国子会社が主として行う事業活動の具体的内容から、法人が事業として行うことができる経済活動の種類に照らして客観的に判断すべきものである」
「当該外国子会社の収入金額の額やその割合、資産の額やその割合のみならず、事業活動において行われる業務の内容及び性質、収入金額及び資産の構成等を考慮することが相当である」
「株式等を保有していることをもって、直ちに株式等の保有を『事業』としているということはできず、当該株式等の保有が事業活動として行われているか否か、他の事業活動との関連性等を考慮して判断すべき」
判断要素の整理
要素本件での評価
収入金額・割合配当が大部分 → 株式保有業的
資産構成株式97% → 株式保有業的
業務の内容・性質地域統括業務を実施 → 統括業的
株式保有の目的グループ経営のため → 統括業的
人的組織統括業務に従事 → 統括業的
下級審との相違
名古屋高裁（原審）
  • 収入・資産構成を重視
  • 形式的に株式保有業と認定
  • → 事業基準を満たさない
最高裁の逆転
  • 事業活動の「具体的内容」を重視
  • 株式保有と統括業務は別事業
  • → 主たる事業は地域統括業務

確立された判断基準
「主たる事業」の判定方法
  1. 事業活動の「具体的内容」を検討
  2. 収入・資産構成は一要素にすぎない
  3. 業務の内容・性質を総合考慮

「株式保有業」の限定解釈
  • 単に株式を保有しているだけでは足りない
  • 株式の売買・運用を目的とする場合
  • 純粋持株会社は該当しうる
  • 事業統括のための保有は別論
平成29年度税制改正との関係
改正法の内容
統括会社については、実体基準・管理支配基準を満たせば事業基準も満たすものとする特例を明文化（措置法66条の6第2項3号ハ）
本判決の意義
  • 改正前の事案に関する判断
  • 改正により立法的に解決された部分あり
  • しかし「主たる事業」判断の一般論は維持
実務への示唆
地域統括会社
統括業務の実態があれば事業基準を満たしうる
立証のポイント
人員配置・業務内容・経営関与の具体的証拠が重要
形式主義の否定
収入・資産構成だけでは判断できない
Key Takeaway：事業基準の判定は、収入・資産の形式的構成ではなく、事業活動の実態を総合的に評価して行う。地域統括会社としての実質的な経営管理機能があれば、株式を多く保有していても「株式保有業」には該当しない。

判旨の分析 - 基準時の厳格解釈
14/15
最高裁判旨（要旨）
「措置法66条の6第1項は、課税対象金額の計算に用いる『請求権勘案保有株式等割合』について、特定外国子会社等の『各事業年度の終了の時』における割合と規定している」
「同項の文言及び趣旨に照らせば、上記割合は、特定外国子会社等の各事業年度の終了の時を基準として判定すべきものであり、その後に行われた増資等による持株割合の変動は考慮しないものと解するのが相当である」
「本件においては、本件SPCの事業年度終了時における上告人の請求権勘案保有株式等割合は100%であり、その後の増資により低下したとしても、課税対象金額の計算においては100%を用いるべきである」
判断の論理構造
論点判断
条文の文言「事業年度終了の時」と明記
基準時の特定事業年度末時点の割合を使用
事後変動の扱い考慮しない
立法趣旨明確な基準時による計算の確実性
原審との相違
東京高裁（原審）
  • 実質的な経済的利益を重視
  • 増資後の割合も考慮すべき
  • → 納税者勝訴
最高裁の逆転
  • 条文の文言を厳格に解釈
  • 事業年度末時点で判定
  • → 国勝訴

先例的価値と実務への影響

確立された法理
基準時の厳格解釈
  • 「事業年度終了の時」は厳格に解釈
  • 事後的な資本変動は考慮対象外
  • 文言に忠実な解釈を採用
計算規定の明確性
  • CFC税制の計算規定は厳格解釈
  • 納税者の主観的意図は無関係
  • 客観的な基準時による判定
税務プランニングへの影響
タイミングの重要性
  • 事業年度末前に資本取引を完了する必要
  • 事後的な調整は認められない
  • 期末時点の持株割合が決定的
スキーム設計上の留意点
  • SPC利用スキームでは期末持株割合に注意
  • 増資・減資のタイミング管理が重要
  • 複数期にわたる計画的な対応が必要
総括：CFC税制最高裁判例の全体像
国勝訴の判例から
  • CFC税制は国家主権の課税権（11302/11344）
  • 租税条約による制約は限定的（11302/11344）
  • 計算規定は文言どおり厳格適用（13900）
納税者勝訴の判例から
  • 外国法人税は実質判断（11342）
  • 事業基準は実態を総合評価（13082）
  • 形式的判断は否定される傾向
Key Takeaway：CFC税制の適用においては、制度全体の合憲性・条約適合性は認められつつも、個別の適用要件については実質的な判断がなされる。計算規定は文言に忠実に解釈されるため、タイミング管理が重要となる。
