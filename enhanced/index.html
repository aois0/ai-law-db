AI向け強化版法令データベース

このデータベースは、LLM/AIエージェントが日本の税法を効率的に検索・参照できるよう最適化されたHTMLファイル群です。

データ構造

ファイル命名規則

パターン例説明
{条番号}.html22.html第22条
{条番号}-{枝番}.html22-2.html第22条の2
index.html-目次（全条文一覧）

HTML属性（data attributes）

各条文には機械可読なdata属性が付与されています：

<article data-law="法人税法" data-article="22">
  <div data-paragraph="1">
    <p>1 内国法人の各事業年度の所得の金額は...</p>
  </div>
  <div data-paragraph="3">
    <p>3 ...次に掲げる額とする。</p>
    <p class="item" data-item="一">一 当該事業年度の収益に係る...</p>
    <p class="item" data-item="二">二 前号に掲げるもののほか...</p>
  </div>
</article>
属性付与要素値の例用途
data-lawarticle"法人税法"法令名の識別
data-articlearticle"22", "22-2"条番号の識別
data-paragraphdiv"1", "2", "3"項番号の識別
data-itemp.item"一", "二", "イ"号・イロハの識別

関連条文リンク

条文中に「政令で定める」「令第○条」などの参照がある場合、関連条文へのリンクが自動生成されます：

<div class="refs">
  <h3>関連条文</h3>
  <ul>
    <li><a href="../hojinzei_seirei/19.html">令第十九条</a></li>
    <li><a href="../hojinzei_kisoku/index.html">施行規則（省令/規則で定める）</a></li>
  </ul>
</div>

AIエージェントでの使用例

1. 特定条文の取得

質問: 法人税法第22条を教えて

URL: https://aois0.github.io/ai-law-db/enhanced/hojinzei/22.html

2. 条文構造の解析

質問: 法人税法第23条第1項第2号の内容は？

手順:

  1. hojinzei/23.html を取得
  2. data-paragraph="1" の div を特定
  3. data-item="二" の p 要素を抽出

3. 委任関係の追跡

質問: 法人税法第23条で「政令で定める」と書かれている部分の詳細は？

手順:

  1. hojinzei/23.html の「関連条文」セクションを確認
  2. リンク先 hojinzei_seirei/ の該当条文を参照

4. CSSセレクタでの条文抽出

// 第22条第3項のすべての号を取得
document.querySelectorAll('[data-article="22"] [data-paragraph="3"] [data-item]')

// 法人税法のすべての条文を取得
document.querySelectorAll('[data-law="法人税法"]')

注意事項

  • 本データは e-Gov法令検索 のXMLデータを加工したものです
  • 附則は現在含まれていません（本則のみ）
  • 法令改正により内容が変わる場合があります
  • 正確な法令解釈には、必ず公式の法令を確認してください
